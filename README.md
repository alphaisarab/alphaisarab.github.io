# alphaisarab

alphaisarab is a simple, **FREE**, unblocked gaming website directed for places that has ~~restrictions~~ on certain, suspicious websites. This project is designed for people who want to **have fun** but have *certain restrictions* on *certain websites* during school, work, etc you get the point.
